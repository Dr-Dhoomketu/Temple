# Shri Ram Mandir — Temple Website

A premium, immersive Hindu temple website inspired by Akshardham Temple (New Delhi).

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS v4
- React Three Fiber + Three.js (3D temple model)
- Framer Motion (animations)
- GSAP

## Features

- Animated temple door opening on load
- Interactive 3D temple model (mouse/touch responsive)
- Dark / Light mode with animated Diya toggle
- Puja timings, Pujari profiles, Donations & Trustees sections
- Fully responsive (mobile + desktop)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build

```bash
npm run build
npm run preview
```

## 3D Model

Place `temple.glb` in `public/models/temple.glb`.
